开启行编辑

发表于 2017-04-12

AG-GRID 开启行编辑

在表头定义的属性中添加editable = true
例如：

{headerName: “Name”, field: “name”, width: 150,editable: true},

如果要开启行编辑，除了editable之外，还需要editType = ‘fullRow’属性

默认情况以及修改

开启编辑的方式默认情况下有以下几种情况：

  • 编辑按键按下，例如enter、delete、等
  • 输入任意字符
  • 鼠标双击
  • 调用 api.startEditingCell()

结束行编辑的默认方式：

  • 回掉stopEditing
  • 聚焦其它单元格
  • 输入enter、desc等建
  • 方向键
  • 窗口关闭
  • 调用gridApi.stopEditing()

修改：

-如果要使用单击，请设置属性：singleClickEdit = true

-单击双击都不可以：suppressClickEdit = true,这个用来放button开启cell编辑时非常有用

-当鼠标失去焦点停止编辑：stopEditingWhenGridLosesFocus = true。
