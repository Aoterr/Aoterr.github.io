jdk8-Lambda表达式规则

发表于 2017-07-05
字数统计 | 阅读时长

Lambda

lambda 表达式的基础语法：java8 中引入了一个 -> 操作符，该操作符将表达式分为两部分

  • 左侧：lambda 的参数列表,匿名内部类中的方法的参数
  • 右侧：表达式中所需执行的功能，即 lambda 体，内部类方法的实现

匿名内部类lambda表达式规则

语法格式——：无参数，无返回值

语法格式二：有一个参数，无返回值

语法格式三：有一个参数，小括号可以省略不写

语法格式四：有两个以上的参数，有返回值，并且有多条语句

语法格式五：若有一条语句，大括号和 return 都可以省略不写

语法格式六：Lambda 表达式参数类型可以省略不写，JVM编译器通过上下文推断出数据类型，即类型推断

左右遇一括号省，左侧推断类型省 能省则省

lambda需要函数式接口的支持，若接口中只有一个方法的接口为函数式接口，可以使用 @FunctionalInterfaace注解修饰一下，检查是否为函数式接口
